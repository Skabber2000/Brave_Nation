Como exportar as imagens

  1. Abra este ficheiro num navegador (Chrome ou Edge recomendados).
  2. Para cada modelo, clique com o botão direito sobre a imagem e selecione Inspecionar.
  3. No painel DevTools, clique com o botão direito no elemento <div class="canvas ..."> e selecione Capture node screenshot.
  4. A imagem será guardada com as dimensões exactas em pixels indicadas em cada modelo.
  5. Alternativa com Playwright: Execute o seguinte comando para capturar todos os modelos automaticamente:
    playwright screenshot --selector=".banner-1" linkedin_banner.html banner-1.png
  6. Alternativa com DevTools: Use Ctrl+Shift+P → "Capture node screenshot" depois de selecionar o nó no painel Elements.
  7. Verifique que as imagens ../img/cover.jpg e ../img/author.jpg estão acessíveis.
