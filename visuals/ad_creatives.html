Como exportar para imagem

  1. Abra esta pagina no Google Chrome ou Microsoft Edge (Chromium).
  2. Abra as DevTools (F12) e aceda a Elements.
  3. Clique com o botao direito no div do anuncio desejado → Capture node screenshot.
  4. O ficheiro PNG sera guardado com as dimensoes exatas (1080×1080 ou 1080×1920).
  5. Alternativamente, use Ctrl+P → Guardar como PDF para versao impressa.
