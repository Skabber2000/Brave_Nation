6.2 Meta Pixel (Important)

Action required: Add Meta Pixel to nacaovalente.com.pt. This enables:
  • Retargeting: Show ads to people who visited the site but didn't RSVP
  • Conversion tracking: Measure outbound clicks to the Google Form
  • Lookalike audiences: Find more people like those who RSVP'd (for future campaigns)
How: Create a pixel in Meta Events Manager → add the base code to <head> of index.html → add an outbound click event on the RSVP button.
